# Reference Dashboard Findings

## Sheets in the reference report:
1. Executive Summary - KPIs overview
2. Clinician Performance - detailed per-clinician stats
3. Insurance Analysis - breakdown by insurance provider
4. Test Analysis - breakdown by test type
5. Monthly Trends - month-by-month referral data
6. Turnaround Analysis - time metrics
7. Patient Demographics - age groups
8. Clinician-Test Matrix - cross-tab of clinician x test
9. Pending Results - tracker for incomplete results
10. Holter Analysis - specific holter monitor analysis
11. Clean Data - raw cleaned data

## Key KPIs from Executive Summary:
- Total Referrals, Referring Clinicians, Total Tests Ordered, Avg Tests/Referral
- Results Completion Rate, Median Referral-to-Appt
- Results Sent, Results Pending
- Median Appt-to-Results, Avg Referral-to-Appt, Avg Appt-to-Results
- Top Insurance Provider

## Insurance Analysis columns:
Insurance Provider, Total Referrals, % of Total, Total Tests, Avg Tests/Referral, Unique Clinicians

## Clinician Performance columns:
Referring Clinician, Total Referrals, Total Tests, Avg Tests/Referral, Results Sent, Results Pending, Completion Rate (%), Avg Referral→Appt (days), Avg Appt→Results (days)
